## PDL-003: Style Calibration Issue Identified

**Date:** October 19, 2025

### Issue/Decision Point

User provided paper excerpts (Introduction, Section IX) and asked: *"consider whether the style needs tuning based on this paper excerpts"*

**Problem Identified by AI:**
- Excessive bolding throughout Appendix (paper uses bold very sparingly)
- Over-enumeration in flowing prose ("First, Second, Third" structures)
- Tone too instructional in places rather than analytical/descriptive
- **BUT:** Appendix IS reference material, should be more structured than argumentative prose

**User Confirmation:** Agreed with assessment, requested:
- Reduced bolding
- Smoother prose in A.2 (graph description)
- Less enumeration in flowing text
- Maintained structure in reference sections (types, tables) where appropriate

### Resolution

**Systematic style revision:**

- Removed most bolding (kept only for critical phase markers, key distinctions)
- Eliminated "First, Second, Third" enumeration in narrative sections
- Changed instructional language ("you must," "the reviewer should") to descriptive ("serves as," "enables," "provides")
- Maintained structured presentation in A.3 (type definitions) and tables
- Improved natural flow particularly in A.2

### Rationale

Appendix serves dual function: must be navigable as reference material (justifies some structure) while maintaining philosophical paper's careful, precise tone. Balance between accessibility and scholarly rigor.

### Impact

Revised Appendix matched paper's style while preserving necessary reference functionality. Natural prose in narrative sections (A.2 graph description), appropriate structure in definitional sections (A.3 types).

## PDL-004: Figure Integration Requirement

**Date:** October 19, 2025

### Issue/Decision Point

User requested fundamental restructuring: *"insert all writing referring to graphs: (1) in square brackets, clearly distinguishing it from text that will remain (2) as a prompt for Claude to create the image in vectorial form"*

**Challenge:** A.2's entire "Conceptual Graph" subsection was written as detailed spatial description intended for someone to sketch from text. This needed transformation into:
- Referenced figure in main text (NOT described)
- Separate comprehensive prompt for vectorial figure creation
- Analytical text discussing what the figure demonstrates

### Resolution

**Approach:**

- Replaced detailed graph description with figure reference: `[Figure 1: Document Creation Flow - See Figure 1 for visual representation...]`
- Rewrote A.2 to **REFERENCE** visual elements rather than **DESCRIBE** them
  - *"As shown in Phase 1..."* instead of *"Show at foundation..."*
  - *"The stacked visual effects in Phase 2 illustrate..."* instead of *"Use stacked effect to show..."*
- Preserved conceptual explanation and interpretation as text
- Removed all spatial/directional drawing instructions from body text

**Created Separately:** Comprehensive Figure 1 prompt (~1,200 words) including:
- Overview and design principles
- Detailed specifications for all 6 phases
- Phase-specific elements, arrows, annotations
- Legend/key requirements
- Style guidelines and size considerations
- Structured for both human designer and AI image generation

### Rationale

Academic papers reference figures rather than embedding descriptions in prose. Separation allows:
- Clean text focused on conceptual significance
- Detailed technical specifications preserved for figure creation
- Independent work on visualization while maintaining analytical discussion
- Professional academic presentation style

### Impact

Transformed A.2 from drawing instructions to proper academic prose that references and interprets visualization. Created standalone figure prompt usable by designer or AI image generation tools.
